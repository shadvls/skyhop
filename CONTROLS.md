# SkyHop — Controls

## Keyboard

| Key | Action |
|---|---|
| Space | Flap / Start / Restart |

## Mouse

| Input | Action |
|---|---|
| Click | Flap / Start / Restart |

## Touch (Mobile)

| Gesture | Action |
|---|---|
| Tap | Flap / Start / Restart |
| Tap (menu) | Select skin / difficulty |

## Tips

- Tap lightly to control bird height
- Avoid pipes and ground
- Score increases by passing through pipes
- Unlock skins by reaching high score milestones
